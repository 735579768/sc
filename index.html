<div class="td-frame" id="td-ud-frame2">
    <div class="td-up">

    <div class="td-line">

<script type="text/javascript">
    var obj = dragFrame.init({
        type: 2,
        mainFrame: '#td-frame',
        frame1: '.td-left',
        frame2: '.td-right',
        frameL: '.td-line',
        callback: function(event) {
            dragFrame.resetSize();
        }
    });
</script>

type拖动类型/方向
类型：int 默认为 1
框架类型1左右 2上下
mainFrame主框架的容器选择器
类型：jquery选择器 默认为：#td-frame
bili框架比例
类型：Array 默认为：[1,1]
minSize框架最小宽/高度
类型：int 默认为：50
frameL拖动线
类型：jquery选择器 默认为：.td-line
frame1第一个框架
类型：jquery选择器 默认为：.td-left
frame1Size第一个框架默认大小
类型：int 默认为：0
frame2第二个框架
类型：jquery选择器 默认为：.td-right
frame2Size第二个框架默认大小
类型：int 默认为：0
callback拖动回调
类型：function 默认为：null
